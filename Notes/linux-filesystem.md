# Linux Filesystem

## What I Learned

### `/`
The root of the entire Linux filesystem.

### `/home`
Where user files and personal projects live.

### `/etc`
System configuration files.

### `/usr`
Installed programs and libraries.

### `/var`
Data that changes over time (logs, databases, caches).

### `/tmp`
Temporary files.

### `/dev`
Represents hardware devices.

### `/proc`
A live view into the running Linux kernel.

## My Takeaway

Linux is organized by purpose instead of putting everything in one place.
